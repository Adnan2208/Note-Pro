# Note-Pro

## Description

***It is a simple web app where you can create an access code and then using that code you can get access to your notes and create new notes.***

## Problem it solves

When working on a shared or temporary machine—such as a college computer—you may not want to log in to GitHub or other online accounts just to access your personal notes.

Note-Pro provides a simple alternative. With just an access code, you can create and access your important notes from anywhere—no account logins required.

### Why Note-Pro

- Access your notes securely using a simple access code
- No need to log in to GitHub or other online services
- Ideal for shared, public, or temporary systems
- Quick and distraction-free note creation and access
